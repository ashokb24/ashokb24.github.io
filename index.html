Portfolio

AWS Serverless Bots

News-Tweet-Bot

The project is hosted on AWS Cloud infra in the form of serverless components. Automated News tweet bot is a stateless software bot which meant to handle the below functionalities on behalf of the twitter user thereby reducing the work of twitter user spending time in opening the twitter app from his mobile app/ desktop application. There is a need to build a automated news tweet bot so that it can be scheduled to invoke the twitter api on a daily basis at a specified time. This bot interacts with open source newsapi to get the latest news from a given news provider, fetches news titles, url associated with the news and publishes to Twitter as a tweet on behalf of user account

AWS Serverless Bots

Auto-Retweet-Bot

The project is hosted on AWS Cloud in the form of Serverless components. Automated Re-Tweet bot is a stateless serverless AWS Application which is meant to search a tweet tweeted by any twitter user with a specific hashtag. The application asks User, Hashtag information from the consumer, integrates with twitter to get the relevant tweet satisfying the condition. Once it finds the tweet, it will re-tweet using the consumer Twitter account and like the tweet.
